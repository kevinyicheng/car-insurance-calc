# 車體險出險影響試算

一個單檔、純前端的小工具，回答客戶最常問的一句話：**「這次出險，明年保費會漲多少？」**

依據「**95.07.14 汽車保險規章費率（金管保二字第 09502070910 號核准）**」的賠款紀錄係數與從人因素係數計算。

線上版：`https://kevinyicheng.github.io/car-insurance-calc/`

---

## 功能

- 輸入被保險人（個人含性別／年齡，或法人公司車）、過去 3 年出險紀錄、今年出險次數
- 即時顯示：目前從人因素係數、明年續保係數、倍率變化、漲幅 %
- 選填本年保費 → 估算明年續保金額與增加金額
- 一鍵「首年投保」情境

---

## 計算邏輯

### 保費公式

| 類型 | 公式 |
|---|---|
| 個人自小客／自小貨（車體險） | 基本保費 × 製造年度費率代號係數 × **（年齡性別係數 ＋ 賠款紀錄係數）** |
| 法人（公司車） | 基本保費 × 製造年度費率代號係數 × **（1 ＋ 賠款紀錄係數）** |

> 關鍵：賠款係數是**加進**年齡性別係數，不是單獨乘 `(1＋係數)`。
> 個人男性 30–60 歲的年齡性別係數剛好是 1.00，此時兩式結果相同——這也是常見的誤會來源。

### 賠款紀錄點數

賠款紀錄點數 ＝ 無賠款年度點數 ＋ 賠款次數點數

| 無賠款年度 | 點數 | | 累計過去 3 年賠款次數 | 點數 |
|---|---|---|---|---|
| 3 年 | −3 | | 1 次 | 0 |
| 2 年 | −2 | | 2 次 | 1 |
| 1 年 | −1 | | 3 次 | 2 |
| 0 年 | 0 | | 4 次 | 3 |

> 賠款第 1 次不加點，第 2 次起每次 +1 點，無上限（4 次以上每增 1 次 +1 點）。

### 點數 → 賠款紀錄係數

每點 ±0.2。點數 −3 → −0.6，0 → 0，+3 → +0.6；3 點以上每增 1 點再 +0.2。

### 年齡性別係數（車體險，95 核定本）

| 年齡 | 男 | 女 |
|---|---|---|
| 未滿 20 | 1.89 | 1.70 |
| 20–25 | 1.74 | 1.57 |
| 25–30 | 1.15 | 1.04 |
| 30–60 | 1.00 | 0.90 |
| 60–70 | 1.07 | 0.96 |
| 70 以上 | 1.07 | 0.96 |

### 滾動視窗

賠款紀錄以「投保當年往前 3 年」為準。明年續保時視窗往前滾一年：丟掉最舊一年，補進今年。工具中「明年點數」即以「2 年前 ＋ 去年 ＋ 今年」計算。

## 邊界案例

| 情境 | 結果 |
|---|---|
| **首年投保** | 無賠款 0 年（從未投保）＋ 賠款 0 次 = 0 點 → 係數 **0**（不加不減）。從人係數 = 年齡性別係數本身。**不是 −0.6**。 |
| 投保滿 3 年且無理賠 | 無賠款 3 年 = −3 點 → 係數 −0.6 |
| 首年投保 + 今年出 1 次 | 明年仍 0 點（第 1 次 0 點）→ 係數 0 |
| 首年投保 + 今年出 2 次 | 明年 +1 點 → 係數 +0.2 |
| 同一賠款係數變動，不同年齡性別 | 漲幅 % 不同（基準 = 年齡性別係數 ＋ 係數，基準越低 % 越大） |

> 工具中「未保」狀態用於表示該年度未投保，不計入無賠款年度。

## 維護注意

1. **係數版本**：年齡性別係數為 95.07.14 核定本。富邦現行版若有微調，更新 `index.html` 內 `AGE_SEX` 對照表後重新部署。
2. **未納入車齡**：製造年度費率代號係數（車齡折舊）未計入，故實收通常略低於試算金額；頁面已標示提醒。
3. **僅供參考**：本工具為業務溝通用試算，非正式報價；正式保費以保險公司核保系統為準。
4. **隱私**：純前端計算，不蒐集、不傳送任何資料。請勿在此 repo 放置保戶個資。
5. **需網路**：此版執行時需連網載入 React。若需離線單檔版，將 React 內嵌即可。
